# 数字方法

提供丰富的数字处理工具函数，包括随机数、范围控制、精度计算、格式化等功能。

## 导入方式

```typescript
import { jqTool } from 'qcpt'

// 使用数字方法
jqTool.helper.number.random(1, 10)
```

## 随机数

### random

生成指定范围的随机整数：

```typescript
jqTool.helper.number.random(1, 10)
// 1-10之间的随机整数
```

### randomFloat

生成指定范围的随机浮点数：

```typescript
jqTool.helper.number.randomFloat(1, 10, 2)
// 1.00-10.00之间的随机数，保留2位小数
```

## 数字范围

### range

生成数字序列：

```typescript
jqTool.helper.number.range(1, 5) // [1, 2, 3, 4, 5]
jqTool.helper.number.range(0, 10, 2) // [0, 2, 4, 6, 8, 10]
```

### clamp

限制数字在指定范围内：

```typescript
jqTool.helper.number.clamp(15, 0, 10) // 10
jqTool.helper.number.clamp(-5, 0, 10) // 0
jqTool.helper.number.clamp(5, 0, 10) // 5
```

## 精确计算

解决 JavaScript 浮点数精度问题：

### add / subtract / multiply / divide

精确加减乘除：

```typescript
jqTool.helper.number.add(0.1, 0.2) // 0.3 (而不是 0.30000000000000004)
jqTool.helper.number.subtract(0.3, 0.1) // 0.2
jqTool.helper.number.multiply(0.1, 0.2) // 0.02
jqTool.helper.number.divide(0.3, 0.1) // 3
```

### round / ceil / floor

精确舍入：

```typescript
jqTool.helper.number.round(1.2345, 2) // 1.23
jqTool.helper.number.ceil(1.2345, 2) // 1.24
jqTool.helper.number.floor(1.2345, 2) // 1.23
```

### toFixed

修复 toFixed 问题：

```typescript
jqTool.helper.number.toFixed(1.2345, 2)
// '1.23'
```

## 百分比计算

### percentage

计算百分比：

```typescript
jqTool.helper.number.percentage(25, 100) // 25
jqTool.helper.number.percentage(1, 3, 2) // 33.33
```

### percentToDecimal / decimalToPercent

百分比与小数互转：

```typescript
jqTool.helper.number.percentToDecimal(25) // 0.25
jqTool.helper.number.decimalToPercent(0.25) // 25
```

## 数字格式化

### padZero

数字补零：

```typescript
jqTool.helper.number.padZero(5, 3) // '005'
```

### toChineseNumber

数字转中文：

```typescript
jqTool.helper.number.toChineseNumber(12345)
// '一万二千三百四十五'

jqTool.helper.number.toChineseNumber(15)
// '十五'
```

### toChineseMoney

数字转大写金额：

```typescript
jqTool.helper.number.toChineseMoney(123.45)
// '壹佰贰拾叁元肆角伍分'
```

## 数字判断

### isEven / isOdd

判断奇偶：

```typescript
jqTool.helper.number.isEven(4) // true
jqTool.helper.number.isOdd(3) // true
```

### isPrime

判断是否为质数：

```typescript
jqTool.helper.number.isPrime(7) // true
jqTool.helper.number.isPrime(8) // false
```

### inRange

判断是否在范围内：

```typescript
jqTool.helper.number.inRange(5, 0, 10) // true
```

## 数学运算

### abs / sqrt / pow

基础数学运算：

```typescript
jqTool.helper.number.abs(-5) // 5
jqTool.helper.number.sqrt(16) // 4
jqTool.helper.number.pow(2, 3) // 8
```

### factorial

求阶乘：

```typescript
jqTool.helper.number.factorial(5)
// 120 (5! = 5×4×3×2×1)
```

### gcd / lcm

最大公约数和最小公倍数：

```typescript
jqTool.helper.number.gcd(12, 8) // 4
jqTool.helper.number.lcm(12, 8) // 24
```
