# JQTooltip 智能提示框

智能提示框组件，支持多行文本省略和自动显示提示。

## 基础用法

```vue
<template>
  <j-q-tooltip content="这是一段很长的文本内容，当文本溢出时会自动显示提示框">
    这是一段很长的文本内容
  </j-q-tooltip>
</template>

<script setup lang="ts">
import { JQTooltip } from 'qcpt'
</script>
```

## 多行文本

```vue
<template>
  <j-q-tooltip :content="longText" :lines="3">
    {{ longText }}
  </j-q-tooltip>
</template>

<script setup lang="ts">
import { ref } from 'vue'
import { JQTooltip } from 'qcpt'

const longText = ref('这是一段很长的文本内容...')
</script>
```

## 自定义样式

使用 `contentStyle` 属性自定义内容样式：

```vue
<template>
  <j-q-tooltip 
    content="这是一段很长的文本内容"
    content-style="width: 200px; color: red;"
    :lines="2"
  />
</template>

<script setup lang="ts">
import { JQTooltip } from 'qcpt'
</script>
```

## API

### Props

| 属性名 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `content` | `string` | `''` | 提示框内容 |
| `lines` | `number` | `1` | 最大显示行数（必须大于 0） |
| `contentStyle` | `string` | `'width:100%'` | 内容样式 |

### Slots

| 插槽名 | 描述 |
| --- | --- |
| `default` | 需要显示提示的内容（默认显示 content） |

## 特性

- 自动检测文本是否溢出
- 只有在文本溢出时才显示 tooltip
- 支持单行和多行文本省略
- Tooltip 内容支持特殊字符换行

## 注意事项

1. **自动检测**：组件会自动检测文本是否溢出，只有在溢出时才显示 tooltip
2. **单行省略**：lines 为 1 时使用单行省略，超出宽度显示省略号
3. **多行省略**：lines 大于 1 时使用多行省略，超出指定行数显示省略号
4. **样式设置**：contentStyle 默认值为 'width:100%'，可以通过该属性自定义样式
5. **实际应用**：常用于表格、列表等场景，自动处理文本溢出
