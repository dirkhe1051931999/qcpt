# 组件概览

QCPT 提供了丰富的企业级组件，分为以下几个类别：

## 表单组件

用于构建表单的输入组件：

- **[JQButton](./form/j-q-button/)** - 增强的按钮组件，支持多种类型、尺寸、加载状态和提示功能
- **[JQButtonDropdown](./form/j-q-button-dropdown/)** - 带下拉菜单的按钮组件，支持权限控制和自定义菜单项
- **[JQInput](./form/j-q-input/)** - 输入框，支持验证、密码切换、敏感信息遮罩
- **[JQSelect](./form/j-q-select/)** - 下拉选择，支持单选/多选、搜索、远程数据
- **[JQDate](./form/j-q-date/)** - 日期选择器，支持单选或范围选择
- **[JQDateMonth](./form/j-q-date-month/)** - 月份选择器，支持单选或范围选择
- **[JQDateYear](./form/j-q-date-year/)** - 年份选择器
- **[JQDatetime](./form/j-q-datetime/)** - 日期时间选择器
- **[JQFile](./form/j-q-file/)** - 文件上传，支持拖拽上传
- **[JQAutocomplete](./form/j-q-autocomplete/)** - 自动完成输入，支持搜索历史记录
- **[JQInputRecognize](./form/j-q-input-recognize/)** - 输入识别，支持自动识别和验证输入内容
- **[JQFormLabel](./form/j-q-form-label/)** - 表单标签，支持必填标记和提示
- **[JQSearchForm](./form/j-q-search-form/)** - 快速搜索表单，带查询/重置按钮
- **[JQPagination](./form/j-q-pagination/)** - 分页组件，用于管理和显示大量数据的分页导航
- **[JCListEditor](./form/j-c-list-editor/)** - 动态列表项编辑器
- **[JCDuoListEditor](./form/j-c-duo-list-editor/)** - 键值对列表编辑器

## 数据展示组件

用于展示数据的组件：

- **[JQTable](./data-display/j-q-table/)** - 数据表格，支持分页、排序、选择
- **[JCTable](./data-display/j-c-table/)** - 轻量级表格组件，支持固定列、省略号提示、复制等功能
- **[JQDetailList](./data-display/j-q-detail-list/)** - 结构化详情展示
- **[JCTree](./data-display/j-c-tree/)** - 树形视图，支持选择
- **[JCTreeSelect](./data-display/j-c-tree-select/)** - 下拉树形选择器
- **[JQTooltip](./data-display/j-q-tooltip/)** - 智能提示框
- **[JCTooltip](./data-display/j-c-tooltip/)** - 灵活的提示框组件，支持多种位置、主题和触发方式
- **[JCTooltipEllipsis](./data-display/j-c-tooltip-ellipsis/)** - 自动检测文本溢出并显示提示的组件
- **[JQPopover](./data-display/j-q-popover/)** - 弹出内容
- **[JCTabs](./data-display/j-c-tabs/)** - 标签页切换
- **[JCTitleLine](./data-display/j-c-title-line/)** - 章节标题分隔线
- **[JCFoldableList](./data-display/j-c-foldable-list/)** - 可折叠列表，支持自动折叠和对话框查看

## 图表组件

基于 Chart.js 构建的数据可视化组件：

- **[JChLine](./charts/j-ch-line/)** - 折线图
- **[JChBar](./charts/j-ch-bar/)** - 柱状图
- **[JChBarLine](./charts/j-ch-bar-line/)** - 混合柱状折线图
- **[JChPieDoughnut](./charts/j-ch-pie-doughnut/)** - 饼图或环形图
- **[JChRadar](./charts/j-ch-radar/)** - 雷达图
- **[JChBubble](./charts/j-ch-bubble/)** - 气泡图

## 工具组件

常用的工具和功能组件：

- **[JQDialog](./utils/j-q-dialog/)** - 对话框，支持动态组件
- **[JQConfirm](./utils/j-q-confirm/)** - 快速确认操作
- **[JQConfirmDialog](./utils/j-q-confirm-dialog/)** - 带图标的确认对话框
- **[JQMessage](./utils/j-q-message/)** - 全局消息通知
- **[JCCopy](./utils/j-c-copy/)** - 复制到剪贴板按钮
- **[JCPermission](./utils/j-c-permission/)** - 基于权限的内容显示
- **[JCEditableProxy](./utils/j-c-editable-proxy/)** - 编辑Proxy
- **[JCFallback](./utils/j-c-fallback/)** - 值回退显示组件，根据值是否存在显示内容或默认占位符
- **[JCFormatLookup](./utils/j-c-format-lookup/)** - 数据格式化显示
- **[JCMoreMenu](./utils/j-c-more-menu/)** - 更多操作菜单
- **[JQOptionGroup](./utils/j-q-option-group/)** - 单选/复选框按钮组
- **[JQNewValue](./utils/j-q-new-value/)** - 支持输入新值的多选下拉框组件
