# 图表组件

QCPT 提供了基于 Chart.js 的图表组件，支持折线图、柱状图、饼图等多种图表类型。

## 折线图 (JChLine)

```vue
<template>
  <j-ch-line 
    :labels="['1月', '2月', '3月', '4月', '5月']"
    :datasets="datasets"
    :height="400"
  />
</template>

<script setup lang="ts">
import { ref } from 'vue'
import { JChLine } from 'qcpt'

const datasets = ref([
  {
    label: '销售额',
    data: [120, 190, 300, 500, 200],
    borderColor: 'rgb(75, 192, 192)',
    backgroundColor: 'rgba(75, 192, 192, 0.2)'
  }
])
</script>
```

## 柱状图 (JChBar)

```vue
<template>
  <j-ch-bar 
    :labels="['A', 'B', 'C', 'D']"
    :datasets="datasets"

</template>

<script setup lang="ts">

const datasets = ref([
  {
    label: '数据',
    data: [12, 19, 3, 5],
    backgroundColor: 'rgba(54, 162, 235, 0.5)'
  }

```

## 混合柱状折线图 (JChBarLine)

```vue
<template>
  <j-ch-bar-line 
    :labels="labels"
    :datasets="datasets"
  />
</template>

<script setup lang="ts">
import { ref } from 'vue'
import { JChBarLine } from 'qcpt'

const labels = ref(['1月', '2月', '3月'])
const datasets = ref([
  {
    type: 'bar',
    label: '柱状图',
    data: [10, 20, 30],
    backgroundColor: 'rgba(54, 162, 235, 0.5)'
  },
  {
    type: 'line',
    label: '折线图',
    data: [15, 25, 35],
    borderColor: 'rgb(75, 192, 192)'
  }
])
</script>
```

</script>

## 通用 Props

所有图表组件都支持以下通用属性：

| 属性名 | 类型 | 描述 |
| --- | --- | --- |
| `labels` | `string[]` | 标签数组 |
| `datasets` | `ChartDataset[]` | 数据集数组（Chart.js 格式） |
| `height` | `number` | 图表高度（像素） |
| `options` | `ChartOptions` | Chart.js 配置选项 |

## 注意事项

- 所有图表组件基于 Chart.js 4.x
- `datasets` 和 `options` 遵循 Chart.js 的标准格式
- 可以通过 `options` 属性自定义图表的详细配置
- 图表会自动响应数据变化
