# QCPT 组件库

基于 Vue 3、TypeScript 和 Quasar 构建的现代化企业级组件库。

## 快速开始

[查看快速开始指南](./quick-start/) - 了解如何安装和配置 QCPT 组件库

[Git 仓库](https://gitlab.redtea.io/redteamobile/ngiot-frontend-cpt/-/tree/dev?ref_type=heads) - 查看源代码

## 特性

- **现代化 UI** - 基于 Quasar 的现代化 UI 组件
- **开箱即用** - 完整的企业级组件，无需额外配置
- **权限控制** - 内置权限控制系统
- **数据可视化** - Chart.js 驱动的图表组件
- **主题定制** - 高度可定制的颜色主题
- **国际化** - 内置中英文支持
- **响应式** - 完美适配各种屏幕尺寸
- **TypeScript** - 完整的 TypeScript 类型支持

## 组件分类

### 表单组件

丰富的表单输入组件，包括输入框、选择器、日期选择器等。

- [JQInput](./components/form/j-q-input/) - 输入框组件
- [JQSelect](./components/form/j-q-select/) - 下拉选择组件
- [JQDate](./components/form/j-q-date/) - 日期选择器
- [JQDateMonth](./components/form/j-q-date-month/) - 月份选择器
- [JQDateYear](./components/form/j-q-date-year/) - 年份选择器
- [JQDatetime](./components/form/j-q-datetime/) - 日期时间选择器
- [JQFile](./components/form/j-q-file/) - 文件上传组件
- [JQAutocomplete](./components/form/j-q-autocomplete/) - 自动完成输入组件
- [JQInputRecognize](./components/form/j-q-input-recognize/) - 输入识别组件
- [JQFormLabel](./components/form/j-q-form-label/) - 表单标签组件
- [JQSearchForm](./components/form/j-q-search-form/) - 快速搜索表单组件
- [JQPagination](./components/form/j-q-pagination/) - 分页组件
- [JCListEditor](./components/form/j-c-list-editor/) - 动态列表项编辑器
- [JCDuoListEditor](./components/form/j-c-duo-list-editor/) - 键值对列表编辑器

[查看所有表单组件](./components/form/)

强大的数据展示组件，包括表格、详情列表、树形组件等。

- [JCTree](./components/data-display/j-c-tree/) - 树形视图组件
- [JCTreeSelect](./components/data-display/j-c-tree-select/) - 下拉树形选择器
- [JCNestedSelect](./components/data-display/j-c-nested-select/) - 嵌套选择器组件
- [JQTooltip](./components/data-display/j-q-tooltip/) - 智能提示框组件
- [JQPopover](./components/data-display/j-q-popover/) - 弹出内容组件
- [JCTabs](./components/data-display/j-c-tabs/) - 标签页切换组件
- [JCTitleLine](./components/data-display/j-c-title-line/) - 章节标题分隔线组件
- [JCFoldableList](./components/data-display/j-c-foldable-list/) - 可折叠列表组件

[查看所有数据展示组件](./components/data-display/)

### 图表组件

基于 Chart.js 的数据可视化组件，支持折线图、柱状图、饼图等。

- [JChBar](./components/charts/j-ch-bar/) - 柱状图组件
- [JChBarLine](./components/charts/j-ch-bar-line/) - 混合柱状折线图组件
- [JChPieDoughnut](./components/charts/j-ch-pie-doughnut/) - 饼图或环形图组件
- [JChRadar](./components/charts/j-ch-radar/) - 雷达图组件
- [JChBubble](./components/charts/j-ch-bubble/) - 气泡图组件

[查看所有图表组件](./components/charts/)

### 工具组件

常用的工具组件，包括对话框、消息通知、权限控制等。

- [JQConfirm](./components/utils/j-q-confirm/) - 快速确认操作组件
- [JQConfirmDialog](./components/utils/j-q-confirm-dialog/) - 确认对话框组件
- [JQMessage](./components/utils/j-q-message/) - 消息通知组件
- [JCPermission](./components/utils/j-c-permission/) - 权限控制组件
- [JCCopy](./components/utils/j-c-copy/) - 复制按钮组件
- [JCEditableProxy](./components/utils/j-c-editable-proxy/) - 双击编辑组件
- [JCFallback](./components/utils/j-c-fallback/) - 异步组件加载占位符组件
- [JCFormatLookup](./components/utils/j-c-format-lookup/) - 数据格式化显示组件
- [JCMoreMenu](./components/utils/j-c-more-menu/) - 更多操作菜单组件
- [JQOptionGroup](./components/utils/j-q-option-group/) - 单选/复选框按钮组组件
- [JQNewValue](./components/utils/j-q-new-value/) - 新增/修改值标记组件

[查看所有工具组件](./components/utils/)

```bash
npm install qcpt
```

然后查看 [快速开始指南](./quick-start/) 了解如何配置和使用。

## 工具函数

丰富的工具函数库，提供对象、数组、字符串、数字等各类处理方法。

- [对象方法](./utilities/object-helpers/) - 对象拷贝、合并、属性操作、转换等
- [数组方法](./utilities/array-helpers/) - 数组去重、树形化、分组、排序、统计等
- [字符串方法](./utilities/string-helpers/) - 脱敏处理、格式化、字符串操作等
- [数字方法](./utilities/number-helpers/) - 随机数、精确计算、数字格式化等
- [URL 方法](./utilities/url-helpers/) - URL 参数处理、URL 构建、编码解码等
- [DOM 方法](./utilities/dom-helpers/) - 剪贴板、文件下载、滚动、元素操作等
- [正则验证方法](./utilities/regex-helpers/) - 手机号、邮箱、身份证等常用验证
- [性能优化方法](./utilities/perf-helpers/) - 防抖、节流、函数缓存、异步控制等

[查看完整工具函数文档](./utilities/)
